# Лицензионные ограничения

## AGPL-3.0

FluffyChat лицензирован под **GNU Affero General Public License v3.0**. FluffyX как форк наследует эту лицензию.

### Обязательства

1. **Open source** — весь исходный код FluffyX должен оставаться открытым
2. **Предоставление исходников** — пользователи приложения должны иметь возможность получить исходный код (ссылка в приложении)
3. **Copyleft** — любые модификации и производные работы также подпадают под AGPL-3.0
4. **Network use = distribution** — если приложение предоставляет сервис по сети, пользователи сервиса также имеют право на исходники

### Практические следствия

- Репозиторий GitHub остаётся публичным
- В приложении (Settings / About) — ссылка на GitHub репозиторий
- Нельзя менять лицензию на проприетарную
- При публикации в App Store / Google Play — ссылка на исходники в описании или в приложении
- Сторонние библиотеки должны быть совместимы с AGPL-3.0

### Совместимость зависимостей

Все текущие зависимости из `pubspec.yaml` используют лицензии, совместимые с AGPL-3.0 (MIT, BSD, Apache 2.0).

При добавлении новых зависимостей — проверять лицензию на совместимость.
